-- 新功能
-- --------
-- 增强字符串加密系统 (五层保护):
-- 
--   L1 - 每字符串独立密钥
--     - 通过 PRNG 从字符串内容派生唯一密钥
--     - 无共享密钥，无法批量解密
--   
--   L2 - 密文切片乱序
--     - 加密后字节切分为4字节块
--     - Fisher-Yates 随机乱序存储
--   
--   L3 - 哈希索引
--     - 源码仅保留8位十六进制哈希键
--     - 不暴露原始字符串内容
--   
--   L4 - 多态解码器
--     - 5种不同解码表达式模板
--     - 运行时随机选择，增加逆向难度
--   
--   L5 - 诱饵解码函数
--     - 插入使用错误密钥/顺序的假解码
--     - 永不调用，增加逆向成本
--
-- 修复
-- ------
-- - 修复 string.format 导致的 Lua 解析错误
-- - 修复 #"xxx" 长度操作符被误提取的问题
-- - 修复 slices 数据结构与解码器不匹配问题
--
-- 改进
-- ------
-- - 字符串池提取逻辑优化
-- - 更安全的 Lua 代码生成方式
